# Key Decisions

> Significant decisions made by gitrails across sessions.
> Used by mirror contradiction-check to detect severity drift.
> Cleared on monthly archive rotation.

## Format

Each significant decision appends:

```
## {date} — {finding_id}
- Decision: {approved|blocked|suppressed}
- Reason: {one-line rationale}
- Agent: {agent}/{skill}
- Severity applied: {CRITICAL|HIGH|MEDIUM|LOW}
```
